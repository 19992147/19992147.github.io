[Codeforces 827E] Rusty String

Posted on 2018-06-04 | In Codeforces | |

Description:
求带通配符字符串的循环节。

Solution:
考虑求循环节的方法，求出$kmp$的$nxt$数组,那么答案就是$n-nxt$。
由于有通配符，所以不能使用$kmp$，于是通过$FFT$计算。两个数组中分别把V和K标记为1，做FFT，如果一个位置的和为0那么意味着这个长度不是循环节，那么长度的约数也不是循环节，于是找出未被标记的长度即可。
