口胡题解

Posted on 2018-07-08 | In 杂 | |

前言

最近没时间写题解，于是便有了这个简易题解，之前省队集训的题解也会在这里更新。

题解

bzoj 4012 开店

动态点分治裸题。每个节点开$vector$，二分即可。

bzoj 4675 点对游戏

考虑每对点对的贡献，设一共有m对幸运数，那么一对点对是幸运的概率是$\frac{2m}{n(n-1)}$,取了x个点的期望是$\frac{x(x-1)m}{n(n-1)}$。长链剖分计算即可。

bzoj 4515 游戏

一个李超线段树，李超线段树每个区间保存覆盖区间最大的线段，每次查询注意左右端点。

bzoj 5336 party

$dp$套$dp$，先$dp$出每种$lcs$状态添加字符后转移到的状态，然后$dp$即可。

bzoj 3639 Query on a tree VII

$LCT$维护黑白两棵树，连通性通过每个点和自己父亲颜色相同连接。$LCT$每个点开$set$维护虚子树最大值。

bzoj 5338 xor

可持久化Trie树

bzoj 5311 贞鱼

设$dp[i][k]$表示前$i$个分成$k$段的最小值。转移$dp[i][k]=min{dp[j][k-1]+sum(j+1,i)}$。
如果对于$k_1<k_2$且$dp[i][k_1]$优于$dp[i][k_2]$,那么以后$k_1$一直优于$k_2$,于是单调队列维护决策单调性，复杂度$O(nklogk)$。
再利用凸优化解决即可,复杂度$O(nlognlogk)$。
看见恰好$k$个可用凸优化，看见$dp$想决策单调性。

bzoj 2827 千山鸟飞绝

动态开点线段树区间赋值。

bzoj 5197 Gambling Guide

设$dp[i]$表示$i$点到$n$的期望钱数，那么考虑更新，$dp[i]$更新相邻的点，如果$dp[i]$更优那么选择$dp[i]$,否则选择$dp[j]$。得出$dp$方程用最短路转移即可。

bzoj 5312 冒险

线段树维护即可，根据势能分析得出复杂度$O(nlognlogw)$。

bzoj 4943 蚯蚓

哈希表维护所有合法子串，根据势能分析二操作最多增加$ck^2$次操作，总串数是$nk$,所以复杂度$O(nk+ck^2)$。

bzoj 5413 color

发现合法情况为一种颜色同时存在于两边或两种颜色同时存在于两边，可以通过斯特林数求出颜色划分。

bzoj 5407 girls

即求三元环以及三元环缺一条边的个数，第一种根号分治，第二种前缀和维护。

bzoj 5408 string

$LCT$维护$parent$树，注意后缀自动机构建。

bzoj 5409 墨水大师

相当于对于每个边双联通分量求答案乘起来，环上方案数可用矩阵快速幂或公式维护，可以打表找规律。由于共有$\sqrt{n}$种环可做到$O(n\sqrt{n})$。

bzoj 5410 分岔路口

这是一种套路题，二分随机跳期望，距离小于期望直接跳，否则随机跳，二分动态树分治即可。
